# Web2App

A reusable Android WebView template to convert any website into an Android app.

## Features

- 🌐 Load any website
- 🚀 Splash Screen
- 🔄 Swipe to Refresh
- ⬅️ Back Navigation
- 🌍 Internet Permission
- ⚡ Fast and lightweight
- 📱 Easy to customize

## Requirements

- Android Studio
- Java
- Android SDK

## How to Use

1. Clone this repository.
2. Open the project in Android Studio.
3. Replace the website URL with your own.
4. Build and run the app.

## Customization

Change the website URL in the WebView configuration.
